# instructions 
*Because said friends can barley operate computers as well*

1) Required: 
   1) [Install the latest version of python](https://www.python.org/downloads/)
2) Optional (I want automatic updates or know how to use git):
   1) Install [Git](https://git-scm.com/)
   2) If you know how to use CMD, Skip this part:
      1) Go to the folder you want this program located in.
      2) Open File explorer
      3) Go to the navigation bar, and click it
      4) type in cmd
      5) the terminal should open
   3) In the terminal, type `git clone https://github.com/Casstaroth/TheGreatAd-dExperiment.git`
3) Git is scary:
   1) download the zip from the github
4) Run the included launch.bat file
